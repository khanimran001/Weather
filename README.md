# 🌤️ Weather App

A simple and responsive Weather App built using **HTML, CSS, and JavaScript** that fetches real-time weather information using the **OpenWeather API**.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- ☁️ Shows weather condition
- 💧 Displays humidity
- 🌬️ Shows wind speed
- ⚡ Real-time weather data using API

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## 📂 Project Structure

```
Weather-App/
│── index.html
│── style.css
│── script.js
└── README.md
```

## 🔑 API Used

This project uses the **OpenWeather API** to fetch real-time weather data.

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder.

3. Replace the API key in `script.js`:

```javascript
const apiKey = "YOUR_API_KEY";
```

4. Open `index.html` in your browser.
